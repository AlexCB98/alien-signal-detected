# Alien Signal Detected

A simple HTML project created to practice basic page structure, images, lists, and links.

---

## Features

- Mission title and date
- Space-themed image
- Equipment list
- Mission objectives
- Google Maps location link

## What I practiced

- HTML boilerplate
- Headings with `<h1>`, `<h2>`, and `<h3>`
- Images with `<img>`
- Unordered lists with `<ul>` and `<li>`
- Links with `<a>`
- Basic HTML indentation
